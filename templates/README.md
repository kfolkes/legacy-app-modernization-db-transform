# Optional scaffold templates

The folders here are **optional**. Neither `/dotnet.modernize` nor `/java.modernize` requires them to complete.

Use them only when Phase 3 of the modernization plan recommends one of these patterns:

| Template | When to use |
|---|---|
| `aspire-orchestrator-template/` | Multi-service .NET 10 solutions composed with .NET Aspire |
| `dotnet-microservice-template/` | Decomposing a monolithic .NET app into microservices |
| `kotlin-bff-template/` | Adding a Kotlin-based BFF (Backend for Frontend) to a modernized stack |

If your modernization plan does not recommend microservices, Aspire, or a BFF, you can ignore this folder entirely.
